# video-boy
A NodeJS utility to download online video

## Quick Start

````javascript
var Youku = require('../lib/youku');
 
var url = 'http://v.youku.com/v_show/id_XOTI0MTE4ODg4.html';
var youku = new Youku();

youku.parse(url, function(e, video){
  // now 'video' contains the real URLs that sotre the video files,
  // feel free to download them!
});
````
